概要

これは R を使ったメタ解析のサンプルコードです。 RStudioで実行することができます。

準備手順

  1. 任意のフォルダを作成します
  2. そのフォルダに
    • sample.R
    • sample.csv
    • report.Rmd
    をダウンロードします
  3. RStudioでそのフォルダを開きます
ダウンロード後、ファイル名は自由に変更できます。

sample.csv の入力

sample.csv には研究データを入力します。
1列目
Study情報
例：Yamada T (2023)
2列目
exposure の eventあり
3列目
exposure の合計
4列目
control の eventあり
5列目
control の合計

入力例

study,event.e,n.e,event.c,n.c
Yamada T (2023),12,40,8,38
Suzuki K (2024),5,20,3,22

sample.R の修正

CSVの名前を変更した場合は sample.R の4行目を修正します。 
dat <- read.csv("sample.csv", stringsAsFactors = FALSE)
例えば 
dat <- read.csv("mydata.csv", stringsAsFactors = FALSE)
また、群名は以下のように設定されています。 
exp_label <- "Group A"
ctrl_label <- "Group B"
これらは任意の名前に変更できます。

report.Rmd の修正

report.Rmd は HTMLレポートを作成するコードです。 以下の項目は変更できます。
  • title
  • author
  • Group A / Group B
例 
title: "Candida auris mortality meta-analysis"
author: "Yukihiro Kaneko"

実行方法

  1. sample.csv にデータを入力
  2. 必要なら sample.R を修正
  3. 必要なら report.Rmd を修正
  4. RStudioで sample.R を実行
  5. report.Rmd を Knit するとHTMLレポートが作成されます

補足

ファイル名を変更した場合は sample.R 側のCSVファイル名も必ず変更してください。

注意

  • 現在のサンプルコードは forest plot（フォレストプロット）の作成のみを行う簡易的なコードです。
  • R および RStudio は別途インストールする必要があります。
  • R や RStudio の基本的な操作方法（スクリプトの実行、パッケージのインストール、R Markdown の Knit など）は、このページでは説明していません。
  • それらの基本的な使い方については、各自で別途学習してください。
